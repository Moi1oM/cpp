dp[i][j] = i번째까지, j번째까지 비교했을 때 최장 길이
그니까 i, j 중 하나라도 0 이면 무조건 0임(초기화 할 때 이미 됨)
s1[i] == s2[j] 라면 dp[i][j] = dp[i-1][j-1] + 1
s1[i] != s2[j] 라면 dp[i][j] = max(dp[i-1][j], dp[i][j-1])


이렇게 하면 길이는 추정되는데, 정확한 배열은 어떻게 넣을거냐?
vector에 길이를 넣는게 아니라 직접 string을 넣으면 안되나?

s1[i].size() == s2[j].size() 라면 dp[i][j] = dp[i-1][j-1] + s1[i] (C++에서 이게 되나?)
s1[i].size() != s2[j].size() 라면 max(dp[i-1][j], dp[i][j-1])
